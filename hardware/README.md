## 📋 集成说明

### 1. SDK文件放置
- 将厂商提供的SDK文件放入对应的 `sdk/` 子目录
- 确保SDK文件权限正确（可执行权限）

### 2. 驱动程序开发
- 每个设备在 `drivers/` 下有独立目录
- 包含Python封装类和配置文件

### 3. 系统适配
- 主要针对银河麒麟V10系统
- 确保兆芯CPU架构兼容性

## ⚠️ 注意事项

1. SDK文件通常较大，建议使用Git LFS管理
2. 部分SDK可能需要厂商授权
3. 驱动集成前请先进行硬件测试
